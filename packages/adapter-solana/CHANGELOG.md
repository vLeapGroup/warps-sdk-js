# Changelog

## [0.2.0-beta.1] - 2024-01-01

### Added
- Initial Solana adapter implementation
- WarpSolanaWallet for transaction signing and wallet management
- WarpSolanaExecutor for transaction creation and execution
- WarpSolanaDataLoader for account and asset data loading
- WarpSolanaSerializer for data serialization
- WarpSolanaExplorer for blockchain explorer URLs
- WarpSolanaOutput for transaction output processing
